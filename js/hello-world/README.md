# My Tsonic Project

This is a sample Tsonic project that demonstrates .NET interop.

## Getting Started

Build and run the project:

```bash
npm run build
./app
```

Or run directly:

```bash
npm run dev
```

## Project Structure

- `src/main.ts` - Entry point
- `tsonic.json` - Project configuration
- `generated/` - Generated C# code (gitignored)
